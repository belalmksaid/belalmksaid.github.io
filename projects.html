PacBot



A maze solving robot for Harvard PacBot Competition 2017. Suprisingly, we won first place. Unlike normal mazes, pacman maze has no "solution", it's a little more involved since we had to account for open loops. I also decided to implement the code on a teenysduino 3.2 instead of a Raspberry Pi Zero which is at least 100 times more powerful.
